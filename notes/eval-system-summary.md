- The evaluation system runs JSONL benchmark tasks through `python3 -m agent_harness.cli eval`, producing JSONL results, Markdown/HTML reports, summary JSON/CSV exports, and per-run transcripts for inspection.
- Tasks can assert expected status, tool usage, required or forbidden answer text, observation text, permission policies, and optional judge/rubric scores; the default suite covers 41 scenarios including tool use, safety, recovery, hallucination, formatting, longer-context, cost-sensitive, and regression cases.
- Reports track success rate, tool accuracy, latency, token usage, estimated cost and cost breakdown, dataset SHA-256, failed checks, recommended fixes, failure categories, provider errors/timeouts, and support provider/model comparisons, determinism checks, trace/replay, and RAG evaluation.
